# 🤖 Basic Chatbot

A simple rule-based chatbot built using Python. It responds to predefined user commands and provides a friendly conversation experience.

## ✨ Features
- User-friendly chatbot
- Personalized greeting
- Random jokes and facts
- Current date and time
- Help menu
- Message counter
- Dictionary-based responses

## 🛠️ Technologies Used
- Python
- datetime
- random

## ▶️ How to Run

```bash
python chatbot.py
```

## 💬 Commands

- hello
- hi
- how are you
- what is your name
- thank you
- good morning
- good night
- who created you
- what can you do
- tell me a joke
- tell me a fact
- date
- time
- help
- exit / bye
